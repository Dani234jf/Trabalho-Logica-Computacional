A era da informática e da tecnologia já está presente nas nossas vidas de diversas formas, principalmente no surgimento de novos recursos e tecnologias. Aprender sobre lógica computacional melhora muito a resolução de problemas tecnológicos dentro de um sistema lógico.

A lógica computacional nada mais é do que a soma entre o raciocínio lógico e o uso de ou computação. Trata-se de um modo de pensar para desenvolver software ou para criar algoritmos.

A base por trás de todos os programas utilizados num computador, telemóvel, tablet ou outros dispositivos é a programação, realizada por meio de linguagens específicas feitas para os programadores conseguirem comunicar-se com as máquinas, como, por exemplo, C# ou Python, que são amplamente utilizadas no mercado entre outras.

A lógica computacional pode contribuir em vários aspetos, mas o principal é ajudar a desenvolver e expandir a forma como pensamos e percebemos o mundo. Isto aplica-se inclusive às tarefas, nas quais os algoritmos estão presentes de certa forma, exigindo um raciocínio lógico para a resolução de vários problemas.

O pensamento computacional, que utiliza a lógica como base, pode ser dividido em quatro fases:

  • A decomposição do problema, separar a situação em várias partes menos complexas;
  • Reconhecimento de padrões: separar as categorias e identificar soluções específicas para cada uma;
  • Abstração: dividir o problema em questões mais simples e acentuadas, para ver além do problema;
  • Algoritmo: criar uma série de etapas para a resolução do problema.

Esta separação em fases ajuda a pessoa a organizar as suas ações e o seu esforço, bem como o tempo gasto para a resolução de um problema. Desse modo, é possível alcançar a eficiência e conseguir resultados mais claros.

A lógica computacional possui várias vantagens, como por exemplo, o desenvolvimento cognitivo, melhoria nos processos de aprendizagem e resolução de problemas, utilizando um pensamento organizado e objetivo.

Variáveis e Operadores

As variáveis são utilizadas para armazenar dados que podem ser manipulados pelo programa. Elas são nomeadas de forma a facilitar a compreensão e manipulação dos dados.

Operadores, por sua vez, permitem realizar operações sobre esses dados, como soma, subtração, comparação, etc. A combinação de variáveis e operadores é o que permite ao programador implementar a lógica desejada.

  int Variavel = 5;
  
  Variavel = Variavel * 2;
  Variavel = Variavel + 5;
  int Variavel = 5;

  Variavel *= 2;
  Variavel += 5;
  // Resultado = 15 
  // Resultado = 20 (com inversão das operações)

Neste exemplo, podemos observar que, na primeira linha, foi criada uma variável chamada Variavel contendo o número 5, com um valor de tipo inteiro int.

A ordem é importante neste caso, pois, se multiplicarmos a variável por 2 depois de lhe adicionarmos 5, o resultado seria diferente.

Condições

As condições permitem que o programa tome decisões baseadas em comparações ou verificações. O comando if/else é uma das estruturas mais fundamentais, onde o programa executa diferentes blocos de código dependendo se uma condição é verdadeira ou falsa. 

  int Variavel = 180;
  string Resultado = "";
  
  if (Variavel > 200) { Resultado = "Maior que 200"; }
  else { Resultado = "Menor que 200"; }
  // Resultado = "Menor que 200"

Este exemplo contém, na primeira linha, a criação de uma variável chamada Variavel com o valor inteiro numérico 180. Na segunda linha, temos uma String chamada Resultado, que não contém nenhum valor. A String é uma variável que armazena texto.

Nas linhas seguintes, estão presentes as condições mais usadas na lógica, que são If e Else. O If contém a condição (Variavel > 200), o que nos indica que a linha à frente, entre parênteses retos, só será executada se a condição for verdadeira. Neste caso, a linha não será executada, pois 180 não é maior que 200.

A próxima linha é composta pelo Else, que é apenas executada quando o If não é executado. Neste caso, esta função (Else) é executada, alterando o valor da variável de texto para (Menor que 200), que é o resultado pretendido.

Loops

Os loops são utilizados para repetir uma série de instruções várias vezes, até que uma condição específica seja atendida. Eles são fundamentais quando é preciso processar grandes volumes de dados ou executar operações repetitivas.

Os loops mais comuns são for e while , cada um com as suas particularidades, mas todos são essenciais para a construção de programas eficientes.

  int Variavel = 0;
  
  for (int i = 0; i < 10; i++) { Variavel += 1; }
  int Variavel = 0;
  
  for (int i = 0; i < 10; i++) // Nota: (i++) => (i = i+1)
  { 
    Variavel += 1; 
  }
  // Variavel = 10   (pela condicao i > 10)
  // Variavel = 100  (pela condicao i > 100)

Neste exemplo, os dois blocos de código fazem exatamente a mesma coisa. A única diferença entre eles é a forma como são escritos, o que muitas vezes facilita a compreensão do código e o simplifica em alguns casos. Como nos outros exemplos, existe uma variável chamada Variavel, do tipo int, com o valor 0.

Dentro do loop, a parte entre parênteses retos (lembrando que ambos os blocos de código têm aparências diferentes, mas são iguais) será executada 10 vezes. Em cada uma dessas repetições, a variável Variavel aumenta 1 valor. Como resultado, após 10 iterações, o valor final de Variavel será 10.

Se mudarmos a condição para (i > 100), o loop irá repetir 100 vezes. Se mudarmos a última parte, para em vez de 1 fazer 2 incrementos (i += 2) na variável i o resultado seria de 50, pois iria de 0 para 2,4,6,8etc, resultando na metade das vezes necessárias para chegar no valor i de 100 e assim acabar o loop.

Funções

Funções em programação são blocos de código reutilizáveis que executam uma tarefa específica quando chamadas. Elas ajudam a organizar e modularizar o código, tornando-o mais eficiente e fácil de entender.

  int Resultado = 0; 

  int Adição(int ValorA, int ValorB)
  {
    return ValorA + ValorB;
  } 
  
  Resultado = Adição(5,6);
  int Resultado = 0;

  int Adição(int ValorA, int ValorB) => ValorA + ValorB ;
  
  Resultado = Adição(5,6);
  // Resultado = 11

Neste bloco de código, na primeira linha, podemos ver uma variável chamada Resultado sendo criada com o valor de 0. Na segunda linha, foi criada uma função chamada Adição. Antes do nome da função, foi colocado int, o que significa que a função vai retornar um valor do tipo int.

No entanto, poderia ser qualquer outro tipo, como bool, string, etc. Se a função não retornasse nenhum valor e fosse apenas uma lista de operações lógicas, poderíamos substituir o int por void. O termo void, que significa nulo, indica que a função não retornará nenhum valor quando for chamada, apenas executará determinadas ações.

Esta função de adição possui dois números como valores de entrada. Neste caso, são dois valores do tipo int, que serão utilizados a seguir. Dentro da função, podemos ver o return, que significa que a função irá retornar um valor, que neste caso é a soma de dois números.

Valores de Entrada e Saída

O input (valor de entrada), no contexto de um programa, refere-se a qualquer dado ou informação fornecida a um sistema ou programa para processamento. O input pode vir de diversas fontes, como utilizadores, dispositivos de entrada, ficheiros ou até mesmo outros sistemas.

Existem vários tipos de input que podem ser utilizados em sistemas de software. Os mais comuns incluem:

  • Dados introduzidos pelo utilizador, como texto digitado em formulários, seleções feitas em menus ou cliques em botões.
  • Ficheiros, como documentos CSV (dados em tabela) ou JSON, que contêm dados estruturados e de fácil leitura.
  • Sensores ou dispositivos, que recolhem informações do ambiente e as enviam para processamento.

A primeira linha cria uma variável chamada inputInt com valor inicial 0. A seguir, há um ciclo while que repete até o utilizador escrever um número válido. Se for possível, o valor é guardado em inputInt, caso contrário, aparece uma mensagem de erro na consola.

Antes do nome da função está long, indicando que o valor devolvido será um número inteiro longo. Dentro da função, um ciclo for calcula o fatorial, multiplicando os números de 1 até ao valor recebido.

O resultado é então mostrado na consola, exibindo o número introduzido seguido do seu fatorial.

Exemplo de Validação do NIF

  bool Nif(string nifNumber)
  {
      string filteredNumber = "";

      for (int i = 0; i < nifNumber.Length ; i++) 
      {
          // discarta / filtra qualquer valor que nao seja um numero
          if ( Char.IsNumber(nifNumber[i]) ) { filteredNumber += nifNumber[i]; }
      }

      // Obter primeiro digito do numero, index 0
      int FirstValue = int.Parse(filteredNumber[0].ToString());

      // Se o numero tiver um comprimento diferente de 9 ou
      // se o primeiro digito for 0, é inválido
      if (filteredNumber.Length != 9 || FirstValue == 0) { return false; }

      int calculoCheckSum = 0;

      for (int i = 0; i < 8; i++)
      {
          calculoCheckSum += (int.Parse(filteredNumber[i].ToString()))*(9 - i);
          // i = 0, calculoCheckSum += 1ºdigit * 9
          // i = 1, calculoCheckSum += 2ºdigit * 8
      }

      int digitoVerificacao = 11-(calculoCheckSum % 11);
              
      if (digitoVerificacao > 9) { digitoVerificacao = 0; }

      // último número do Nif de index 8 na string (nº9 digito)
      int digitoVerificaçãoNif = int.Parse(filteredNumber[8].ToString());

      // Se o digito de verificação calculado for igual ao do Nif é válido
      if ( digitoVerificacao == digitoVerificaçãoNif ) { return true; }
      else { return false; }
  }

Código retirado da wikipedia (modificado) Número de identificação fiscal

A primeira linha cria uma função chamada Nif, que retorna um valor bool (verdadeiro ou falso). O propósito desta função é validar um número de NIF (Número de Identificação Fiscal).

A função recebe como entrada uma string chamada nifNumber, que contém o número de contribuinte. A primeira coisa que a função faz é percorrer os caracteres da string e guardar apenas os dígitos numéricos numa nova string chamada filteredNumber. Qualquer carácter que não seja número é descartado.

De seguida, a função garante que o número tem 9 dígitos e que o primeiro dígito não é 0. Se não cumprir estas condições, a função retorna false. Depois, a função calcula um valor chamado calculoCheckSum somando os primeiros 8 dígitos, multiplicados por pesos específicos que vão de 9 até 2.

Com base nesse valor, é determinado o dígito de verificação (último dígito do NIF). Se o dígito de verificação calculado for maior que 9, é ajustado para 0.Por fim, a função compara o dígito de verificação calculado com o último dígito do NIF. Se forem iguais, a função retorna true, indicando que o NIF é válido. Caso contrário, retorna false.
